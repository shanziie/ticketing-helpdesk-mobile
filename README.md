# E-Ticketing Helpdesk Mobile App

## 📌 Description
A mobile-based E-Ticketing Helpdesk application developed using Flutter. 
This application is designed to handle user complaints, ticket management, and support communication efficiently.

## 🎯 Purpose
This project is developed based on Software Requirement Specification (SRS) for academic purposes.

## 👥 User Roles
- User: Create and track tickets
- Helpdesk: Manage and respond to tickets
- Admin: Monitor and control the system

## ⚙️ Features
- User Authentication (Login, Register, Logout)
- Create Ticket (with image/file upload)
- Ticket Tracking & Status Monitoring
- Comment / Reply System
- Notification System
- Dashboard (Ticket Statistics)
- Ticket History

## 🛠️ Technology Stack
- Flutter (Frontend)
- REST API (Backend)
- Firebase / Supabase (optional)

## 📱 Screens
- Splash Screen
- Login Screen
- Dashboard
- Ticket List
- Ticket Detail
- Create Ticket
- Profile
